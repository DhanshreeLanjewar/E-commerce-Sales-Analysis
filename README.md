
This project performs E-commerce Sales Data Analysis using Python.
It analyzes sales data to identify total revenue, best-selling products, and monthly sales trends, and presents insights using visualizations.
The project follows a complete data analysis pipeline:
Load data
Clean data
Analyze data
Visualize results
Generate insights

Objectives:
Analyze sales data using pandas
Handle missing and inconsistent values
Calculate key business metrics
Create at least two different types of charts
Present clear insights from the analysis
